AI Action Assistant

A production-grade Personal AI Assistant platform — built from scratch with Python, FastAPI, and Groq LLaMA 3.3 70B. This is not a chatbot wrapper. It is a full agentic AI system that executes real-world tasks through natural language.

VERSION 3.0.0 · PRODUCTION

📖 Project Overview

This system enforces a strict architectural principle: The LLM handles reasoning only. Python code executes all real-world actions.

The AI never pretends to send an email — only the Gmail API sends emails. The AI never pretends to create a calendar event — only the Google Calendar API creates events. Every action goes through a validation layer before execution. Every irreversible action requires explicit user confirmation via a preview screen.

The LLM cannot hallucinate email addresses — they are regex-verified against actual user messages. The LLM cannot hallucinate dates — all dates are validated as parseable and in the future.

⚡ Implemented Services

📧 Email Service

Send emails via Gmail API with OAuth2. Full draft → preview → confirm → send workflow with anti-hallucination validation.

📅 Calendar Service

Schedule events on Google Calendar. Validates dates, times, and timezones before creating events.

🌤 Weather Service

Real-time weather data for any city worldwide via OpenWeatherMap API.

📰 News Service

Live news headlines on any topic from NewsAPI with source attribution and timestamps.

🔍 Web Search

Internet search with DuckDuckGo + Wikipedia fallback for comprehensive results.

📄 Document Summarizer

Summarize URLs, PDFs, DOCX, XLSX, and TXT files using Groq LLM with intelligent chunking.

🧠 RAG Knowledge Base

ChromaDB vector store with Sentence Transformers for semantic search and context-aware responses.

💬 Conversation Memory

Custom rolling buffer with LLM compression for maintaining context across long conversations.

✨ Key Features

🔒 User Authentication: JWT-based auth with bcrypt password hashing and per-user Google OAuth2 tokens
🛡 Anti-Hallucination Validation: Custom Python validators prevent AI from fabricating emails, dates, or executing unverified actions
✅ Confirmation Gates: All irreversible actions (email, calendar) require explicit user approval with full preview
💾 Persistent Sessions: Chat history stored in PostgreSQL with per-user session management
📁 File Upload: Support for PDF, DOCX, XLSX, TXT with automatic summarization
🔄 MCP Protocol: 9 tools exposed via Model Context Protocol for Claude Desktop and Cursor IDE
⚡ Rate Limiting: In-memory per-session rate limiting to prevent abuse
🎨 Modern UI: Responsive design with no framework dependencies — pure HTML/CSS/JS

💡 Why Use This Platform?

1. Production-Ready Architecture: Unlike demo projects, this system is built with real-world deployment in mind — proper auth, rate limiting, error handling, and database persistence.

2. True Agentic AI: The AI doesn't just chat — it executes real tasks. Send actual emails, create real calendar events, fetch live data from the internet.

3. Safety First: Every action is validated. The AI cannot hallucinate recipients, dates, or execute unverified commands. You always see a preview before anything irreversible happens.

4. Extensible Design: Clean separation of concerns makes it easy to add new services. Each service is a standalone module with its own validation logic.

5. Fast Inference: Groq's LLaMA 3.3 70B delivers ~200 tokens/second — responses feel instant compared to traditional LLM APIs.
